## Getting Started

### Prerequisites

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/herdianurdin/dicoding-react-pemula.git
```

2. Install NPM packages

```sh
npm install
```

## Usage

### Run Project

```sh
npm run start
```

### Build Project

```sh
npm run build
```
